# 인공지능 모델 개발 기획서

## Ⅰ. 기획서 개요

### 1. 분야 및 문제 해결 접근 방식
본 기획서는 공정거래위원회 의결서 기반 지능형 하이브리드 RAG 및 법무 컴플라이언스 지원 시스템 개발을 제안합니다. 대규모 법률 문서의 복잡한 법리적 맥락을 정밀하게 파악하기 위해 공식적으로 제공된 하이브리드 데이터를 기반으로 한 검색 체계를 구축하는 것을 목표로 합니다. 단순 키워드 검색의 한계를 극복하기 위해 의미 중심의 밀집 검색 기술인 Dense Retrieval 기술과 법률 용어 및 조항 번호에 특화된 희소 검색 기술인 Sparse Retrieval 기술을 결합한 하이브리드 전략을 채택합니다. 특히 엄격한 민감도 분석 과정을 거쳐 최적의 모델 조합을 선정함으로써 조사 생산성을 극대화하고자 합니다.

### 2. 예상 모델 구조 및 최적화 방법론
1. BGE-M3 및 SPLADE 그리고 BM25 등 다양한 후보군을 대상으로 한 하이브리드 검색 민감도 분석 실시
2. LangGraph의 StateGraph를 활용한 질문 의도 분류 및 최적 검색 경로 할당 지능형 에이전트 워크플로우 설계
3. BGE-M3 ColBERT 리랭커를 포함한 다양한 재정렬 모델의 성능 교차 검증 수행
4. 최종 결과 생성을 위해 8B 파라미터 이상의 규모를 가진 언어 모델 중 최적의 모형 선정을 위한 테스트 진행

### 3. 인공지능 결과 내용 요약
실제 평가 환경에서의 신뢰성을 보장하기 위해 Gemini 3 Flash Preview 모델을 활용하여 원천 데이터에서 추출한 고품질 합성 질문 및 정답 세트 기반의 객관적 벤치마크를 수행합니다. Recall@5 및 Mean Reciprocal Rank 그리고 BERTScore 등 다각적 지표를 통해 모델 변종 간의 성능 격차를 수치화합니다. 체계적인 민감도 분석 과정을 거쳐 공정거래위원회 법률 문서의 특성에 가장 부합하는 검색 및 생성 조합을 식별하고 도메인 특화 데이터에 최적화된 하이퍼파라미터 구성을 확보하여 검색의 정밀도와 생성의 신뢰성을 동시에 달성할 예정입니다.

## Ⅱ. 기획서 상세내용

### 1. 분석 및 인공지능 모델 상세내용

#### 가. 의결서 전처리 및 청킹 방법
1. kiwipiepy 형태소 분석기를 활용하여 법률 고유명사 및 조항 번호에 대한 정밀 토큰화 수행
2. 의결서 제목과 피심인 그리고 위반유형 메타데이터를 청크 텍스트 맨 앞에 접두어 형태로 결합하여 문맥 단절 문제 해결
3. 공식 하이브리드 JSON의 논리적 청크 구조를 그대로 수용하여 법리 문맥의 왜곡을 방지하고 정답 데이터와의 정렬성 확보

#### 나. 검색 및 리랭킹 상세 파이프라인
1. BGE-M3 기반의 밀집 검색과 SPLADE 기반의 희소 검색을 결합한 하이브리드 인덱스 구축
2. 키워드 매칭 정밀도를 높이기 위해 SPLADE에 높은 가중치를 부여하는 Reciprocal Rank Fusion 알고리즘 적용
3. 리랭킹 단계에서 BGE-M3 ColBERT 모델을 활용하여 상위 후보군의 순위를 재정렬함으로써 검색 품질의 안정성 확보

#### 다. 모델 최적화 워크플로우
1. 법률 도메인 특화 형태소 분석 및 메타데이터 융합 전처리 실시
2. 후보 임베딩 모델 기반의 벡터 인덱싱 및 성능 비교 실험 수행
3. LangGraph 기반의 지능형 에이전트를 활용한 질문 의도 파악 및 라우팅 적용
4. 8B 파라미터 이상의 고성능 언어 모델을 활용한 최종 법리 요약 답변 생성

### 2. 결과 해석 및 시사점
1. 다차원 민감도 분석을 통한 기술 최적화 과정에서 청크 크기와 검색 가중치 알파 그리고 Reciprocal Rank Fusion 가중치 등 핵심 하이퍼파라미터를 변수로 설정하여 성능 변화 추이를 정밀하게 관측합니다.
2. 공정거래위원회 법률 문서의 특성인 고밀도 정보와 복잡한 참조 체계에 최적화된 하이브리드 검색 및 ColBERT 리랭킹 조합을 데이터 기반으로 도출하였습니다.
3. 민감도 분석을 통해 도출된 최적의 검색 및 생성 조합이 단일 모델 대비 정답 재현율 및 답변 정확도 면에서 유의미한 성능 향상을 보임을 실증적으로 입증하였습니다.
4. 단순한 기술 도입을 넘어 실험적 데이터를 바탕으로 공정거래 법률 도메인에 가장 부합하는 독자적인 기술 스택을 체계적으로 식별하여 시스템의 완성도를 극대화하였습니다.

### 3. 기대효과
1. 실증 테스트로 검증된 최적 RAG 시스템 구축을 통해 공정거래위원회 심사관의 의결서 분석 및 정보 검색 효율을 극대화하여 조사 행정의 혁신을 달성합니다.
2. 합성 데이터 벤치마크에서 검증된 고정밀 생성 모델을 채택하여 국민에게 정확하고 신뢰할 수 있는 법률 요약 정보를 제공함으로써 공공 서비스의 질을 제고합니다.
3. 민감도 분석으로 확보된 안정적인 성능을 바탕으로 소상공인 및 기업의 법 위반 자가 예방을 효과적으로 지원하여 사회적 가치를 창출합니다.
